# 🌫️ AQI Prediction System

## 📌 Overview
This project predicts the Air Quality Index (AQI) based on environmental pollutant levels using Machine Learning. It also includes data visualization and an interactive Streamlit web application.

---

## 🚀 Features
- Machine Learning model (Random Forest Regressor)
- Predicts AQI based on pollutants
- Data visualization (graphs & trends)
- City-wise pollution analysis
- Interactive Streamlit web app
- Real-time AQI prediction UI

---

## 🧠 Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn
- Streamlit
- Joblib

---

## 📊 Input Features
- PM2.5
- PM10
- NO2
- CO
- SO2

---

## 🎯 Output
- Predicted AQI value
- Air quality category:
  - Good 😊
  - Moderate 😐
  - Poor 😷
